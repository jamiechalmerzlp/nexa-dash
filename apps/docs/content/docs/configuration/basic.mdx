## `NEXADASH_WIDGET_LIST`

Selects which widgets to show on the dashboard.

The available options are: `os`, `cpu`, `storage`, `ram`, `network`, `gpu`.

- type: `string (comma separated list)`
- default: `os,cpu,storage,ram,network`

| `os,cpu,storage,ram,network`                               | `ram,cpu,storage,network`                                                     |
| ---------------------------------------------------------- | ----------------------------------------------------------------------------- |
| ![Dashboard in normal order](/img/screenshot_darkmode.png) | ![Dashboard in alternate order](/img/screenshot_darkmode_alternate_order.png) |

## `NEXADASH_PORT`

The port where the express backend is running (the backend serves the frontend, so it is the same port for both).

- type: `number`
- default: `3001`

## `NEXADASH_ROUTING_PATH`

The path where NexaDash is being served. This is useful if you want to run NexaDash behind a reverse proxy on a sub-path
(e.g `https://example.com/nexadash` would need a value of `/nexadash`).

- type: `string`
- default: `/`

## `NEXADASH_PAGE_TITLE`

If you want to show a custom string in the browser page title.

- type: `string`
- default: `NexaDash`

## `NEXADASH_DISABLE_INTEGRATIONS`

Disables support for integrations. This does two things: disable CORS and disable open API endpoints.

- type: `boolean`
- default: `false`

## `NEXADASH_DISABLE_BACKGROUND_STATS_COLLECTION`

Disables background collection of metrics.

- type: `boolean`
- default: `false`

## `NEXADASH_SHOW_NEXADASH_VERSION`

If you want to show the version number of your NexaDash instance on the bottom right of the page, or alternatively on the GitHub icon (hover).

The available options are: `bottom_right`, `icon_hover` and `none`.

- type: `string`
- default: `icon_hover`

| `bottom_right`                                                                                            | `icon_hover`                                                                                                      |
| --------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| ![NexaDash version bottom right](/img/version_bottom_right.png) | ![NexaDash version on GitHub icon hover](/img/version_server_hover.png) |

## `NEXADASH_USE_IMPERIAL`

Shows any units in the imperial system, instead of the default metric.

- type: `boolean`
- default: `false`

| `false`                                                                                                   | `true`                                                                                                                  |
| --------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| ![CPU widget with temps in celsius](/img/cpu_view_temps.png) | ![CPU widget with temps in fahrenheit](/img/cpu_view_temps_fahrenheit.png) |

## `NEXADASH_ALWAYS_SHOW_PERCENTAGES`

To always show the current percentage of each graph in the top-left corner. Without enabling this,
they will only be shown on lower resolution devices (mobile phones).

- type: `boolean`
- default: `false`

| `false`                                                                                                      | `true`                                                                                                             |
| ------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------ |
| ![Dashboard iwith no percentages](/img/screenshot_darkmode.png) | ![Dashboard with percentages](/img/screenshot_darkmode_with_perc.png) |
