## Options

### `NEXADASH_OS_LABEL_LIST`

Selects which data is shown in the server widget.

The available options are: `os`, `arch`, `up_since`, `nexadash_version`.

- type: `string (comma separated list)`
- default: `os,arch,up_since`

### `NEXADASH_SHOW_HOST`

If you want to show the host part in the server widget (e.g. `NexaDash` -> `jamiechalmerzlp.github.io/nexa-dash`).

- type: `boolean`
- default: `false`

| `false`                                                                                           | `true`                                                                                                  |
| ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
 | ![Server widget normal](/img/server_view_normal.png) | ![Server widget with host](/img/server_view_with_host.png) |

### `NEXADASH_CUSTOM_HOST`

If you want to show a custom host in the server widget (needs [`NEXADASH_SHOW_HOST`](#nexadash_show_host) enabled to work).

- type: `string`
- default: `unset`

## Styles

### `NEXADASH_OS_WIDGET_GROW`

To adjust the relative size of the OS widget.

- type: `number`
- default: `2.5`

### `NEXADASH_OS_WIDGET_MIN_WIDTH`

To adjust the minimum width of the OS widget (in px).

- type: `number`
- default: `300`

## Overrides

### `NEXADASH_OVERRIDE_OS`

- type: `string`
- default: `unset`

### `NEXADASH_OVERRIDE_ARCH`

- type: `string`
